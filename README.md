This repository is a personal project for collecting, processing, and analyzing patent data from USPTO. The code is organized around fetching and cleaning patent records, generating analytical outputs, and producing figures for exploratory review.

Key points:
- Designed for personal use rather than production deployment.
- Supports patent data ingestion, processing, and basic analysis workflows.
- Includes tools for generating reports and visualizations from cleaned patent datasets.
- The NLP component is still early-stage and needs a lot of work before it can be relied on for robust text analysis.

Overall, the repository is intended as a research and development workspace for exploring the patent prosecution field and the companies involved.
